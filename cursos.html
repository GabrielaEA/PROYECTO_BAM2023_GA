Progamación web basica

  • HTML

    • HTML es el acrónimo de HyperText Markup Language (Lenguaje de Marcas de Hipertexto), que es el lenguaje de programación utilizado para crear páginas web.
    Al aprender HTML, podrás:

    • Crear estructuras y diseños para páginas web.
    • Insertar texto, imágenes, videos y otros tipos de contenido en una página web.
    • Agregar hipervínculos y enlaces a otras páginas web y sitios.

  • css3

    • CSS3 es una poderosa herramienta para diseñar elementos HTML. Aquí hay algunas cosas que puedes aprender con CSS3:

    • CSS3 le permite diseñar elementos utilizando una amplia variedad de propiedades, incluyendo fuente, color, fondo, margen, relleno, borde y más.
    • Con CSS3, también puede crear animaciones, transiciones y efectos que hagan que su sitio web sea más interactivo y atractivo.
